Step 2 — Base Score Formula
75 + (Offense TSR × 3) − (Opponent TSR × 2)

Calculate this before rolling for each team separately.

Step 3 — D20 Scoring Table

Roll one d20 per team. Add the modifier to the base score.

Step 4 — Overtime

If scores are tied after regulation, both teams roll again. Apply the same formula but divide the final result by 8 (round up) and add to the regulation score. Repeat if still tied.

Example
Duke (TSR +3) vs Siena (TSR −1)
Duke base: 75 + (3×3) − (−1×2) = 86
Siena base: 75 + (−1×3) − (3×2) = 66
Duke rolls 12 (+2) → 88  |  Siena rolls 8 (−4) → 62
Duke wins 88–62
